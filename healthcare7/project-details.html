Cognitive Wellness and Personalized Medical Guidance Ecosystem

About this Project

This project focuses on providing personalized medical guidance through an interactive chatbot interface. Users can ask about diseases, symptoms, and general health queries, receiving real-time assistance.

Key Features

  • Real-Time Health Guidance: Immediate responses to user queries.
  • Disease Prediction: AI-driven analysis based on symptoms.
  • Nearby Hospitals and Doctors: Assistance in locating medical facilities.
  • Mental Health Support: Resources for stress and anxiety management.
  • Cost-Free Access: Free healthcare guidance for wider accessibility.
  • User-Friendly Interface: Designed for users of all technological backgrounds.

Technologies Used

  • HTML, CSS, JavaScript for Frontend
  • JSON Dataset for Health Conditions
  • MongoDB for Database
